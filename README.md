# Rishi Ecommerce Purchase Project

Welcome to the Rishi Ecommerce Purchase Project! This repository is dedicated to analyzing e-commerce purchase data and visualizing it using various data analysis tools.

## Repository Description

This repository includes:

1. **Dataset Overview:**
   - **File path:** `Ecommerce Purchases.csv`
   - The dataset contains information about e-commerce transactions, such as address, purchase price, browser information, credit card details (obfuscated), and other relevant attributes.

2. **Notebook Analysis:**
   - **File path:** `Rishi_ecommerce_purchase_.ipynb`
   - A Jupyter Notebook that performs exploratory data analysis (EDA) to understand trends, behavior, and key insights from the dataset.

## Features and Functionalities
- Analysis of e-commerce purchase data.
- Insights into customer behavior, purchase patterns, and transaction attributes.
- Trend analysis in purchase price, browser type usage, and transaction timings (AM/PM).
- Data visualizations using Python libraries such as Matplotlib and Seaborn.
